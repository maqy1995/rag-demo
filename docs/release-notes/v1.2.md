## 新增（Added）

### 代码模块
- **`src/rag_demo/llm/`**（`base.py` + `openai_compat.py` + `__init__.py`）
  - `BaseLlmClient` / `BaseEmbedder` 抽象基类（frozen dataclass config，构造时校验）
  - `OpenAICompatibleClient(LLMConfig)` / `OpenAICompatibleEmbedder(EmbedConfig)` 单类覆盖 4 家
  - 错误映射：401/403 → `AppError(GENERATE_LLM_FAIL)`（http_status=e.status_code）；429/5xx → 默认按 retry 策略重试 → 耗尽后 `AppError`；连接错误 → 503
  - Retry 退避基数 0.5s（测试场景 monkeypatch 到 0.001s）
- **`src/rag_demo/chunker.py`** —— 标题+长度混合切分
- **`src/rag_demo/vector/__init__.py`** —— FAISS `IndexFlatIP` + JSON metadata 持久化
- **`src/rag_demo/vector/faiss_store.py`** —— `add(vectors, metas)` / `search(query_vec, top_k)` / `save(dir)` / `load(dir)`

### 脚本 & 工具

- **`scripts/eval_recall.py`** —— Recall@K 评估 CLI + `pytest -m eval` 入口

## 修改（Changed）

- **`src/rag_demo/retrieve.py`** —— 从 stub 切真：加载 FAISS + embed query + search + 转 `Hit`（保持 v1 签名）
- **`src/rag_demo/generate.py`** —— 从 stub 切真：`_call_llm` 走 `BaseLlmClient.stream()` 流式；模块级 `set_llm_client()` / `get_llm_client()` 注入
- **`src/rag_demo/ingest.py`** —— 从 stub 切真：chunker + embedder + FAISS 写盘；保留 NB2 building 中间态 + NB3 fallback
- **`src/rag_demo/errors.py`** —— 加 `EMBEDDING_FAIL` 错误码
- **`src/rag_demo/config.py`** —— 默认 provider 从 ollama → openai；`generate.llm.provider` 与 `generate.embedding.provider` 独立字段
- **`.env.example`** —— 4 个 `*_API_KEY` 字段（分别命名 `OPENAI_API_KEY` / `ZHIPU_API_KEY` / `MIMAX_API_KEY` / `MIMO_API_KEY`）
- **`pyproject.toml`** —— extras 重新组织（llm / vector / web meta-extras）；移除 ollama 相关依赖（v1.1 时代保留的 `langchain` / `llamaindex` 仍可选但默认不装）
- **测试** —— 80 → **145 passed**（净增 65 断言）

## 移除（Removed）

- ❌ **本地 Ollama 不再是默认 provider**（owner 拍板，MAQ-25 评论）
- ❌ **Ollama fallback 路径** —— 无 API key 时 fast-fail `AppError(401, GENERATE_LLM_FAIL)`，不再回退本地
- ❌ **`generate.llm.provider: "ollama"` 默认值** —— 默认改为 `openai`
- ❌ **stub 行为** —— `retrieve` 不再永远返回空 list；`generate._call_llm` 不再返回固定前缀
- ❌ **`init_static.py` 占位 HTML 生成** —— 改为提交 `static/index.html` 真实 UI 到 git
- ❌ **`scripts/init_static.py`**（如未来不再使用）—— 待 MAQ-44 后续清理（v1.2 暂保留兼容）

## Quick start（v1.2 完整版）

```bash
git clone https://github.com/maqy1995/rag-demo
cd rag-demo
uv sync --extra llm --extra vector --extra web --extra dev
cp .env.example .env
# 编辑 .env 填 1-4 个 *API_KEY
cp config.example.yaml config.yaml
# 编辑 config.yaml 设 vault.path / 选 LLM provider / 选 Embedding provider

# 启动 (主入口)
uv run rag-demo up
# 浏览器开 http://127.0.0.1:8000/
```

详见 [README §快速上手（5 分钟）](../README.md#快速上手5-分钟)。

- ❌ `ollama` 不再是默认 provider

## 性能预期（远程 API）

| 笔记规模 | 冷启动 ingest | 检索 P50 | 端到端问答 P50 |
|----------|--------------|----------|----------------|
| **100 篇**（≈ 5000 chunks） | **5-15 min** | < 100 ms | 1-3 s |
| **500 篇**（≈ 25000 chunks） | 25-60 min | < 200 ms | 1-3 s |
| **1000 篇**（≈ 50000 chunks） | **30-90 min** | < 500 ms | 1-3 s |
| **5000+ 篇** | 不推荐 | 500 ms-2 s | 1-3 s（**先换 HNSW 或切换 Chroma**） |

详见 [README §性能预期](../README.md#性能预期)。
